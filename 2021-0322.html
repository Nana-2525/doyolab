1.【gitの初期設定】

git用のフォルダの作成

まずgit用のフォルダを作りたいと思います。

Macの人はターミナルを立ち上げ、Windowsの人はGit Bashを立ち上げてください。

立ち上げたターミナルorGitBashに下記のコマンドを入力するとフォルダが作成されます。

※mkdirはmake directory(ディレクトリ(フォルダ)を作る)の略です

mkdir

gitの設定

gitの初期設定を行います。下記のコマンダーをターミナル(or GitBash)に入力してEnterを押しましょう

git init

次に下記のコマンドを入力し、Enterを押します。※自分のメールアドレスには、既存のアカウントで使用しているメールアドレスを入力しましょう。

git config --global user.email"自分のメールアドレス"

次に下記のコマンドを入力し、Enterを押します。※自分の名前には、既存のアカウントで使用しているものを使いましょう。

git config --global user.name"自分の名前(アルファベット)"

2.既にあるGitHubのアカウントに新しくリポジトリーを作成する

GitHubにログインする。GitHub ログインページ

GitHub右上に表示されているアイコンの左側の+↓マークをクリックし、New repositoryをクリックし、新しいリポジトリーを作ります。

Repository nameにdoyolabと入力しdoyolabというリポジトリーを作成します。

次に、PublicとInitialize this repository with a READMEに✓を入れます。

✓を入れた後にCreate Repositoryを押して下さい。

次に、リポジトリーのSetting内のGitHub Pagesまで移動します。

GitHub PagesのSourceの項目をmainに変更してください。変更した後Saveボタンを押して下さい。

するとYour site is published at URL(自分のHPのURL)が表示されます。※このURLが

3.アクセストークンを作成

ここを参考にアクセストークンを作成します。

個人のアクセストークンは作成時しか見ることができないので、コピーして保存しておいてください。

4.リポジトリーを自分のPCにクローン

gitのURLをコピー

githubのdoyolabというリポジトリーの中の緑色でCodeと書かれたボタンを押し、リポジトリーのURLをコピーします。

mygitに移動

ターミナル(or GitBash)で、下記のコマンドを入力しmygitフォルダに移動します。※cdはchange directoryの略です。

cd mygit

自分のPCにリポジトリーをクローンする

ターミナル(or GitBash)で下記のコマンドを入力し、GitHubリポジトリのクローンを作成してください。(URLは上でコピーしたURLです)

git clone リポジトリのURL

確認

mygitの中にdoyolabというフォルダができて、doyolabフォルダの中にREADME.mdファイルが入っていれば成功です。

5.htmlフォルダの作成

まずmygitの中のdoyolabフォルダに下記の名前のファイルを作成してください。

・css

・images

次に、テキストエディタを開いて、ファイル→名前を付けて保存→名前をindex.htmlにしてdoyolabのフォルダに保存

公開

下記の手順でアップロードしてください。

cd mygit/doyolab(doyolabフォルダに移動するだけなので、既に移動済みの場合はいらない)

git add .

git commit -m"メッセージ"

git push
